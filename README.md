&#x20;Amazon Sales Analysis



&#x20;Project Overview



This project analyzes Amazon sales data to identify sales trends, customer behavior, and business insights using Python, Pandas, SQL, and data visualization techniques.



&#x20;Objectives



\* Clean and preprocess raw sales data

\* Perform exploratory data analysis (EDA)

\* Identify sales patterns and trends

\* Generate business insights and recommendations

\* Build dashboards and visualizations



&#x20;Dataset



Amazon sales dataset containing information about orders, products, sales, and customer transactions.
